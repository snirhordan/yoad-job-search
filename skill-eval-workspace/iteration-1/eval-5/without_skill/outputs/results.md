## Recommendations for Yoad

### Best Platforms to Focus On
1. **LinkedIn** (set up job alerts for "Python Developer" in Tel Aviv, filter by "Entry Level")
2. **AllJobs** (most popular Israeli job board, many Python positions)
3. **Drushim** (strong for tech jobs, good junior filtering)
4. **GotFriends** (specialized tech recruiter, personal service)
5. **Glassdoor** (good for salary research + applications)

### Job Search Strategy
1. **Apply broadly:** True "junior Python developer" postings are limited. Also search for:
   - "Backend Developer" (many use Python)
   - "Software Developer" + Python
   - "Automation Developer" + Python
   - "Data Engineer" (Python-heavy)
   - "DevOps Engineer" (often uses Python scripting)

2. **Target companies with training programs:** Some companies (like the cybersecurity firm on JobMaster) offer positions with no experience required and include intensive training.

3. **Consider the greater Tel Aviv area:** Herzliya, Ramat Gan, Petah Tikva, and Raanana are all within commuting distance and have many tech companies.

4. **Leverage CS degree:** A fresh CS degree is highly valued in Israel's tech market. Highlight:
   - Algorithms and data structures knowledge
   - Any Python projects or coursework
   - GitHub portfolio
   - Familiarity with frameworks (Django, Flask, FastAPI)

5. **Register with tech recruiters:** GotFriends, SQLink, WOJO, Dialog, and Matrix all specialize in placing developers and can match you with suitable positions.

6. **Attend meetups:** Tel Aviv has an active Python/tech community. Check meetup.com for Python Israel events.

### In-Demand Skills to Highlight
- Python (core)
- SQL / Databases (PostgreSQL, MongoDB)
- REST APIs / FastAPI / Django / Flask
- Docker / Containers
- Git / Version Control
- Cloud basics (AWS preferred in Israel)
- Linux
- Data structures & algorithms
- Any ML/AI exposure (growing demand)
